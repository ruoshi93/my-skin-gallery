## 🚀 启动方式

1. 克隆本仓库：
   ```bash
   git clone git@github.com:ruoshi93/my-skin-gallery.git
2. 安装依赖：
   ```bash
   npm install
3. 启动本地开发环境：
   ```bash
   npm run dev
4. 打开浏览器访问：
   ```bash
   http://localhost:3000
